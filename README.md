ini adalah projeck mata kuliah pengembangan beckend
1.pertemuan Belajar menbuat projeck sederhana dengan express 



cara membuat projeck :
1.buat folder baru
2.masuk kedalam folder dengan perintah cd nama folder dari terminal
3.kenalkam projeck dalam folder dengan perintah npm init unutuk membuat package.jsho isi sesuai dengan identitas projeck diakhiri dengan ketikan (yes)
4.install dependenci/package express dengan perintah(npm i express)
5.buat file index.js untuk file utama projeck"sever ada disini"
6.untuk menjalanakan gunakan perintah "node indexk.js"
7.tunggu proses dan kemudian buka browser dan ketikkan http://localhost:port
